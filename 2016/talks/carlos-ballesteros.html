JTransc: Java and Kotlin everywhere through Haxe

In this talk Carlos will explain what is JTransc, how does it works and why he did it. Also he will explain how to integrate with haxe libraries and how to embed plain haxe code. He will show the current workflow and will talk about the future of the project.

  • What is JTransc
  • How does it works (overview)
  • Productivity over writting haxe directly
  • Sharing code with java backends
  • Why JTransc instead of other alternatives like: GWT, RoboVM, TeaVM, plain kotlin js
  • Why target to Haxe
  • How to integrate with haxe libraries
  • Demos
  • Future
by Carlos Ballesteros

Interested in programming and videogames since I was a child, I grew up grateful to people that shared their code before I had direct access to the Internet. After that I started to create tons of opensource projects and libraries in several different languages. Most of them related to videogames, emulators, generic libraries or productivity. I'm interested in both: backend and frontend development.
